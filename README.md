# Data Structures and Algorithms (DSA)

This repository contains implementations of various data structures and algorithms in Python, focusing on fundamental concepts and efficient implementations.

## Queue Implementations

This repo currently includes three different implementations of a Queue data structure:

- **queue_using_circular_array.py**: A queue implemented using a circular array for efficient space usage and O(1) enqueue/dequeue operations.
- **queue_using_doubly_linked_list.py**: A queue implemented using a doubly linked list, allowing for dynamic sizing and O(1) operations at both ends.
- **queue_using_singly_linked_list.py**: A queue implemented using a singly linked list, with O(1) enqueue and O(1) dequeue operations.

## Usage

Each implementation provides standard queue operations:
- `enqueue(item)`: Add an item to the back of the queue
- `dequeue()`: Remove and return the item from the front of the queue
- `is_empty()`: Check if the queue is empty
- `size()`: Get the number of items in the queue
